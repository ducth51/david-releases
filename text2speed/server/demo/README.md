# Thư mục demo

Mỗi mẫu demo là một **cặp file cùng tên**:

```
<tên giọng>.txt    ← câu văn hiển thị ở cột "Text"
<tên giọng>.wav    ← audio phát ở cột "Sample"
```

`/api/demo/list` chỉ liệt kê những giọng có **đủ cả hai** file, nên file `.txt`
mồ côi sẽ không xuất hiện trên giao diện.

Cách tạo nhanh: mở app, dán câu trong file `.txt`, chọn model tương ứng, bấm
**Generate** rồi **Download Audio**, đổi tên file tải về thành `<tên giọng>.wav`
và bỏ vào thư mục này.
